# LLM4SE Research Skills

This repository collects Claude Code/Codex skills for software engineering research with large language models. The goal is to provide reusable research workflows for LLM4SE work such as literature surveys, single-paper analysis, research topic discovery, CCF-A paper writing, rebuttal preparation, and experiment metric verification.

## Planned Skills

`llm4se-literature-survey` supports multi-paper literature surveys, related-work mapping, source reliability checks, formal publication status confirmation, and gap synthesis.

`llm4se-paper-analysis` supports close reading of one paper or a small fixed set of papers, including problem framing, method decomposition, experiment analysis, contribution boundaries, limitations, and citation usability.

`llm4se-topic-discovery` supports research idea evaluation for high-quality software engineering venues, with emphasis on real SE pain points, prior-work gaps, technical novelty, evaluation feasibility, baselines, and paper story.

`ccfa-paper-writing` supports CCF-A-oriented manuscript writing and revision, including motivation, related work, method description, claim-evidence alignment, and polished paper-ready wording.

`ccfa-rebuttal-guidance` supports reviewer-comment analysis, response strategy, weakness triage, evidence-backed rebuttal drafting, and reviewer-by-reviewer response organization.

`experiment-metric-verification` supports checking experimental metrics, benchmark versions, dataset splits, baselines, table consistency, artifact evidence, and result claims.
